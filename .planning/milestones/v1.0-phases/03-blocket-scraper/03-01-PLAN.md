```javascript
function classifyCategory(keyword) {
  const kw = keyword.toLowerCase();
  if (/gpu|rtx|gtx|rx\s?\d|grafik|geforce|radeon/.test(kw)) return 'gpu';
  if (/cpu|ryzen|intel|core\s?i\d|processor|xeon/.test(kw)) return 'cpu';
  if (/ram|ddr\d|minne/.test(kw)) return 'ram';
  if (/ssd|hdd|nvme|m\.2|storage|lagring/.test(kw)) return 'storage';
  return 'unknown';
}
```

**BlocketScraper class:**

`scrape(keywords)`:
- Loop keywords sequentially with `await sleep(REQUEST_DELAY_MS)` between fetches (skip delay before first keyword)
- Collect all listings into allListings array
- After loop: deduplicate by listing id using a Set (same keyword may appear twice if user passes duplicates)
- Canary check: if `allListings.length === 0 && keywords.length > 0`, call `logger.warn({ keywords }, 'Blocket canary FAIL: zero results — possible structure change')`
- Return deduplicated listings

`_fetchKeyword(keyword)`:
- Construct URL: `${SEARCH_URL}?q=${encodeURIComponent(keyword)}`
- Fetch with undici `fetch()` — set headers:
  - `User-Agent`: `'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36'`
  - `Accept-Language`: `'sv-SE,sv;q=0.9,en;q=0.8'`
  - `Accept`: `'text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8'`
- If `!res.ok`: `logger.warn({ status: res.status, url }, 'Blocket fetch non-200')` and return []
- On network error: `logger.error({ err: err.message, url }, 'Blocket fetch error')` and return []
- Call `this._parse(html, keyword)`

`_parse(html, keyword)`:
- `cheerio.load(html)` then iterate `$('a.sf-search-ad-link')`
- For each anchor element:
  1. `listingId = $a.attr('id')` — skip if falsy or already in seenIds Set
  2. `url = $a.attr('href')` — skip if url does not include `/recommerce/forsale/item/`
  3. `fullUrl = url.startsWith('http') ? url : BASE_URL + url`
  4. `$a.find('span[aria-hidden="true"]').remove()` then `title = $a.text().trim().substring(0, 250)` — skip if empty
  5. `$article = $a.closest('article')`
  6. `priceText = $article.find('div.font-bold span').first().text().trim()`
  7. `price_sek = parsePrice(priceText)` — skip if null
  8. Push Listing: `{ id: 'blocket:' + listingId, marketplace: 'blocket', title, price_sek, url: fullUrl, category: classifyCategory(keyword), listingType: 'buy_now', auctionEndsAt: null }`
- `logger.info({ keyword, count: listings.length }, 'Blocket scrape complete')`
- Return listings array

**CRITICAL — DO NOT:**

- Import anything not already in package.json
- Add pagination (Phase 3 accepts first-page results only)
