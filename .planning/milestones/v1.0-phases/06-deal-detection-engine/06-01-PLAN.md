Implementation details:
- Regex: `/^(\d{1,2})\s+(\w+)\s+(\d{2}):(\d{2})$/` on trimmed raw string
- Map monthStr.toLowerCase() to SWEDISH_MONTHS index; return null if undefined
- Construct `new Date(year, monthIndex, day, hour, minute, 0, 0)` using now.getFullYear()
- Year-boundary fix: if `candidate.getTime() < (nowMs ?? Date.now()) - 60_000`, set year +1
- TWO_HOURS_MS = 2 * 60 * 60 * 1000; pass if `endsAt - nowMs <= TWO_HOURS_MS` AND `endsAt > nowMs`
- Use `module.exports = { parseAuctionEnd, passesAuctionFilter }`
- CommonJS ('use strict') — no ESM
